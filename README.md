# Mega Man X4 Randomizer / MZ-X4-Randomizer
Randomizer for Mega Man X4 for Playstation.

For use with the NTSC North American version. Click Help for detailed information.

Click [Releases](https://github.com/MottZilla/MZ-X4-Randomizer/releases) to download the latest version. There may be a Beta build with new features to try. If you have problems with the latest beta try the newest non Beta release instead.
